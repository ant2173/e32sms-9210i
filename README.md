# E32SMS for Nokia 9210i

An AI-assisted port of SMS Plus (Sega Master System emulator; Version 1.3  by Charles MacDonald) to the Nokia 9210i, developed and tested on original hardware.

## Current status

Private working repository. Source code, build instructions, hardware tests and development notes are being prepared for publication.

## Target platform

- Nokia 9210i Communicator
- Symbian OS 6.0
- Series 80
- ARM9 / EKA1

## Development model

Project direction, experiment design, hardware testing, log collection and result validation are performed on original hardware.

Code and documentation are developed with AI assistance and accepted only after practical testing.

## Repository status

This repository is not ready for public use yet.
