## Development journal

The five-part development journal is available in both languages:

- [Read in English](docs/en/devlog/README.md)
- [Читать по-русски](docs/ru/devlog/README.md)

The series follows the project from its first hardware output at roughly 0.1 FPS to a profiled 15.2 FPS build. It covers EKA1 writable-static-data restrictions, the `.app` + `.exe` architecture, direct framebuffer access and hardware-verified performance work.
