# Управление с клавиатуры Nokia 9210i

[English version](../../en/engineering/keyboard-input.md)

## Предмет отчёта

Документ описывает первый рабочий путь ввода в E32SMS: события физической клавиатуры Nokia 9210i преобразуются в состояние контроллера Sega Master System, которое использует SMS Plus.

Сборка ARMI UREL проверена на оригинальном устройстве прохождением первого уровня Sonic the Hedgehog.

## Стратегия разработки

Ввод разделён на два слоя:

1. фиксированная раскладка для доказательства базового механизма;
2. настраиваемое переназначение, которое будет добавлено позже.

Это позволило не отлаживать одновременно события клавиатуры и интерфейс настроек.

## Определение кодов клавиш

Физические коды были измерены на устройстве, а не выведены из предположений о символьных константах Symbian.

Диагностический обработчик записывал в лог каждое нажатие и отпускание. Клавиши нажимались в известном порядке, что дало аппаратно подтверждённую карту для Up, Down, Left, Right, Button 1, Button 2 и Pause.

Когда окончательная раскладка будет закреплена, точную числовую таблицу стоит добавить рядом с реализацией.

## Связь с состоянием контроллера

В SMS Plus команды Master System представлены битами состояния контроллера:

- событие нажатия устанавливает нужный бит;
- событие отпускания снимает его.

Первая фиксированная раскладка использует стрелки Nokia для движения, две соседние клавиши для двух кнопок действий и Enter для Pause.

## Аппаратная проверка

Проверены:

- все направления;
- обе кнопки;
- Pause;
- корректное отпускание;
- отсутствие перепутанных направлений;
- прохождение первого уровня и загрузка второго.

Тем самым подтверждён полный путь от физической клавиатуры Series 80 через события Symbian к портам эмулируемого контроллера.

## Текущее ограничение

Раскладка пока зашита в код. Интерфейса настройки нет.

Эмулятор продвигает игру примерно на 27 кадров в секунду, поэтому вся игра идёт приблизительно на 45% штатного темпа 60 Гц. Ввод работает, но недостаток скорости заметен при игре.

## Атрибуция и проверка лицензии

Подход к обработке событий адаптирован из порта gnuboy для Series 60.

Если использовалась копия или близкая адаптация исходного кода, а не только изучение идеи, точные upstream-файлы, copyright notice и лицензию нужно добавить в `THIRD_PARTY_NOTICES.md`, а обязательные уведомления сохранить в затронутых исходниках.
